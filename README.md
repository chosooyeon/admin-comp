# Admin Component Library

관리자 페이지 구축을 위한 컴포넌트 라이브러리입니다. Next.js 15와 TypeScript를 기반으로 제작되었으며, 재사용 가능한 관리자 UI 컴포넌트들을 제공합니다.

## 기술 스택

- Next.js 15
- TypeScript
- Tailwind CSS
- React
- ESLint

## 시작하기

개발 서버 실행:

```bash
npm run dev
# or
yarn dev
```

http://localhost:3000 에서 결과를 확인할 수 있습니다.

## 주요 기능

- 반응형 디자인
- 접근성 고려
- 다크모드 지원
- TypeScript 기반 타입 안정성
